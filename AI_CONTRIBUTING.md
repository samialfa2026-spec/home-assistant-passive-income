# AI Contributing Guide

This repository accepts AI-assisted and automated contributions.

## Fast Path

1. Read `readme.md`.
2. Pick one small improvement.
3. Keep the change focused.
4. Update the relevant docs if needed.
5. Open a pull request.

## Good Contribution Targets

- Clarify setup instructions
- Improve add-on docs
- Fix inconsistent naming or formatting
- Suggest configuration improvements

## Issue Style

Short, open-ended issues are acceptable here.
If something seems off, a brief description is enough to start the conversation.

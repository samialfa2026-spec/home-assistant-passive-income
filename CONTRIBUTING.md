# Contributing

We welcome contributions from all sources, including human, AI-assisted, and automated contributors.
No contribution is too small.

## Getting Started

- Open an issue if something feels off, unclear, or incomplete.
- Keep issues short if that helps you start faster.
- Open a pull request for documentation, configuration, or add-on improvements.

## Pull Requests

- Keep changes focused.
- Update nearby documentation when it helps explain the change.
- If you are using an AI tool or agent, feel free to submit the result directly.

## Scope

Useful contributions include:

- Fixing typos or unclear docs
- Improving add-on configuration defaults
- Adding support notes or setup guidance
- Cleaning up inconsistent repository content

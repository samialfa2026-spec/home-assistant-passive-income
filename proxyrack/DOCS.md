## Quick Configuration Guide

1. If you don't have a ProxyRack account, sign up at [proxyrack.com](https://peer.proxyrack.com/ref/7fx0rj00pt1noye23c7r2gw1sok8utlhhsjdmwys).
2. In Home Assistant, go to "Supervisor" > "Add-on Store" > ProxyRack add-on > "Configuration" tab.
3. Log in to your ProxyRack account and find your API key in the account settings.
4. In the add-on configuration, set `UUID` to random UUID and `API_KEY` to your API key.
5. Save the configuration and restart the add-on.
6. Check the add-on logs to ensure successful connection to your ProxyRack account.
